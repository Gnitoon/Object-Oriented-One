## Usage

```js

// instance
const oneObj = new One();

// get the value
const one = oneObj.getOne();

// use it!
i = one + i;

// OR use directly
i = new One().getOne() + i;

// extra: setOne(<value>) (value must be 1)
oneObj.setOne(1)

```
